# Introduction

## About this Workshop

In this lab, you will learn using pretrained models, you can explore how to create vector indexes, generate vectors for queries and similarity searches, optimise performance with indexing and approximate searches, and monitor and report system performance.

Estimated Workshop Time: 1 hour and 30 minutes

### Objectives

In this workshop, you will learn how to:
* Convert pretrained models to ONNX model
* Load Embedding Model into Oracle Database
* Create and Populate a Relational Table for PDF Data

### Prerequisites
This lab assumes you have:
- An Oracle account
